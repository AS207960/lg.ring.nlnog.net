NLNOG Looking Glass


This Looking Glass is hosted by NLNOG, The Dutch Network Operators Group. The Looking Glass runs OpenBGPD as backend and a custom made front end, for which the source code can be found on GitHub. The code was inspired on bird-lg by Mehdi Abaakouk.

The Looking Glass is managed by the NLNOG RING Administrators which can be reached at ring-admins@nlnog.net. You can contact them for questions, reporting issues or setting up new BGP sessions with the Looking Glass server.

Hosting of this Looking Glass is generously sponsored by BIT.nl.
